变量

变量本质：程序在内存中申请的一块用来存放数据的空间

语法 ： var 变量名;

JS中一般使用单引号

声明多个变量只需写一个 var，多个变量之间用英文逗号隔开.

变量一般都是声明之后在使用， 尽量不要使用name来作为变量名

数据类型

使用var 声明变量的时候， 只有当给这个变量确定值之后，才根据确定的值来推断它的数据类型

JS是一种动态语言，不必首先确定变量的数据类型

JS数据类型分为两大类：

简单数据类型（Number， String ，Boolen，Undefined，NULL）；

复杂数据类型 （object）

Number数据类型

Number 数字型，包含整数和小数

isNaN（） 是用来判断一个数是否为非数字，返回一个结果，如果是非数字，那么返回true，否则返回false

字符串型String

可以是 单引号也可以是双引号，在JS中推荐使用单引号

字符串长度： length属性获取整个字符串长度 string.length

字符串拼接：多个字符串之间使用 + 进行拼接，其拼接方式为 字符串 + 任何类型 = 拼接之后的新字符串

+号口诀：数值相加，字符相连

boolean以及undefined和null

boolean类型 返回的是真或假（true或false），true和false可以参与运算，true当1来看，false当0来看*

undefined 未定义数据类型 ，和数字相加，最后的结果是NaN（非数值）

null空类型，表示空值 学到对象时再细讲

typeof检测变量数据类型： typeof + 变量名 – typeof age；

数据类型的转换

使用的表单、prompt获取的数据默认是 ：字符串类型， 通俗的来说，就是把一种数据类型的变量转换成另外一种数据类型

有三种方式的转换：转换为字符串类型 、转换为数字型、转换为布尔型

重点掌握第三种： 加号拼接字符串转换

掌握parseInt 和 parseFloat这两种

运算符

运算符（operator）也被称为操作符，是用于实现赋值、比较和执行算术运算等功能的符号

算术运算符

尽量不用浮点数进行运算，会造成精度丢失问题, 也不能之间拿浮点数进行相比较是否相等

表达式：由数字、运算符、变量等组成的式子

递增递减运算符

**前置递增运算符 ++ num ** ++num = num + 1; 先加加值然后再运算

后置递增运算符 num++ num++ = num +1; 和前置递增的区别在于： 先执行运算之后，再将值加1

比较运算符

== 等于符号 ，默认转换数据类型 会把字符串的数据类型转换为数字型 ： 18 == ‘18’ 是相等的 只要求值相等

=== 全等符号，要求值 和 数据类型都一致 才是相等的

逻辑运算符

逻辑中断-逻辑与： 如第一个表达式为真，则返回第二个表达式； 如果第一个表达式为假，则返回第一个表达式

赋值运算符： = 、+=、-=、*=、/=

运算符优先级

函数

通过函数封装一些大量重复的代码可以使程序更简洁，每个函数实现一定的功能，使用的时候，直接调用

函数声明：

function是声明函数的关键字，必须小写 ， 通过调用函数来执行函数里面的代码

函数的参数

形参可以看做是不用声明的变量，如果传进去的实参数小于形参，多余的形参会存在但没有接受值，为一个undefined类型

如果实参多于形参，会取实参的个数，多余的不会被接收

**return 关键字 – 函数返回值 ** ，return终止函数并且只能返回一个值 , 如果用逗号隔开多个值，以最后一个值为准

如果函数没有return 返回的是undefined值，未定义的值

arguments的使用

只有函数才有 arguments 对象，而且是每个函数都内置好了这个 arguments

作用域

JavaScript的作用域： 代码名字在某个范围内起作用和效果, 目的是为了提高程序的可靠性更重,要的是减少命名冲突

全局作用域：在整个script标签 或者是一个单独的js文件

局部作用域：在函数内部就是局部作用域 这个代码的名字只在函数内部起作用

全局变量：在全局作用域下，在全局下都可以使用

局部变量：在局部作用域下的变量，在函数内部的变量就是 局部变量

注意：函数的形参也是局部变量

if、for语句中的{ } 不是块级作用域，也就是说在 { } 定义的变量是全局变量 不是局部变量，可以外部调用

作用域链：内部函数访问外部函数的变量，采取的是链式查找来决定变量取哪个值（就近原则）这种结构我们称之为作用域链
