HTML基础

HTML

HTML:超文本标记语言（hypertext Markup Language）；描述网页的一种语言

网页后缀：html

网页是由：图片、链接、文字、声音、视频等元素组成；其实就是一个html文件

Web标准

Web标准是由 W3C组织和其他标准化组织制定的一系列标准的集合。W3C（万维网联盟）

Web标准的构成：结构（Structure）、表现（Presentation）、和行为（Behavior）

HTML基本语法

双标签关系可以分为：包含关系和并列关系

HTML基本结构标签

根据标签的语义，在合适的地方给一个最为合理的标签，可以让页面结构更清晰

标题标签:

< h1> - < h6> : 总共六级标签

段落标签：

< p > - < /p> : 给文字进行分段

换行标签：

< br> : 换行标签，是一个单标签 原本单词为：break

文本格式化标签：

< div> 和 < span>标签

< div> 和 < span> 是没有语义的，它们就是一个盒子，用来装内容（布局）

div最显著的特点是一个div只占一行,span都可以放在一行上

图像标签和路径

1、图像标签：< img> 标签用来定义HTML页面中的图像。

图像的插入需要把插入的图片放入与文件相同的路径下

相对路径

绝对路径

绝对路径直接复制图片所在位置地址 来进行访问

超链接标签

在HTML标签中，**< a >**标签用于定义超链接，作用说从一个页面链接到另一个页面

1、超链接的语法格式

_self用当前窗口打开，为默认值， _blank在新窗口打开，一般用_blank

锚点链接：点我们点击的链接，可以快速定位到页面中的某个位置

在链接文本的href属性中，设置属性值为**#名字**的形式；如： 第2集 < /a>

找到目标位置标签，里面添加一个id属性 = 刚才的名字， 如： < h3 id = “two”> 第2集简介< h3>

注释

<! – 注释语句 –> ；快捷键： ctrl + /

特殊字符

&nbsp； 空格符 、 &lt； 小于号、 &gt； 大于号

表格标签

表格主要用于显示、展示数据，不是用来布局页面的

表头单元格：也是一个单元格，常用于表格的第一行，突出重要性，表头单元格里面的文字会加粗居中显示 语法：< th> 文字 < /th>

表格属性

表格属性在实际开放中不常用，通过CSS来设置

表格结构标签

< thead> < /thead> ; 用于定义表格的头部。< thead>内部必须有< tr>标签。一般是位于第一行

< tbody>< /body> ; 用于定义表格的主体。主要用于放数据主体

合并单元格

跨行合并：rowspan = “合并单元格个数” 跨列合并：colspan = “合并单元格个数”

列表标签

表格是用来展示数据的，那么列表就是用来布局的

列表的特点：整齐、整洁、有序，它作为布局会更加自由和方便

根据使用情景不同，列表可以分为三大类：无序列表、有序列表和自定义列表

表单标签

使用表单目的是：收集用户信息

在HTML中一个完整的表单通常是由表单域、表单控件（也称为表单元素）和 提示信息3个部分组成

表单域

表单域是包含表单元素的区域

语法: < form action = “url地址” method = “提交方式” name = “表单域名称”>

表单控件

input输入表单元素

type类型：

text(文本框) 、password（密码框） 、radio（单选按钮）、checkbox（复选框）、

submit（提交按钮）：把表单域form 里面的表单元素 里面的值 提交给后台服务器

**reset(重置按钮)**：重置按钮还原表单元素的初始状态

button（按钮）：搭配JavaScript启动脚本使用

file（文件域）：上传文件使用

radio单选按钮必须有相同的名字，才可以实现多选一

input其它属性

< label>标签

为input元素定义标注（标签），点击label标签内的文本自动转到对于表单元素上

语法：< label for = “sex”> 男 < input type = “radio” name = “sex” id = “sex”>

核心：< label>标签的 for属性应当与相关元素的id属性相同

< select>下拉表单元素

在页面中，如果有多个选项让用户选择，并且想要节约页面空间时，可以使用select标签定义下拉列表

语法：< select > < option > 选项1 < /option> < option> 选项2 < /option> < option> 选项3 < /option> < /select>

注意：select中 至少包含一对 < option>；在< option>中定义selected = “ selected ”时，当前页面为默认选项

< textarea>文本域元素

语法：< textearea row = “3” cols = “20”> 文本内容 < /textarea>

通过< textarea>标签可以轻松的创建多行文本输入框； cols = “每行中的字符数” ， rows = “显示行数”

实际开放中通过CSS来改变大小
