HTML

HTML:超文本标记语言（hypertext Markup Language）；描述网页的一种语言

网页后缀：html

网页是由：图片、链接、文字、声音、视频等元素组成；其实就是一个html文件

Web标准

Web标准是由 W3C组织和其他标准化组织制定的一系列标准的集合。W3C（万维网联盟）

Web标准的构成：结构（Structure）、表现（Presentation）、和行为（Behavior）

HTML基本语法

双标签关系可以分为：包含关系和并列关系

HTML基本结构标签

根据标签的语义，在合适的地方给一个最为合理的标签，可以让页面结构更清晰

标题标签:

< h1> - < h6> : 总共六级标签

段落标签：

< p > - < /p> : 给文字进行分段

换行标签：

< /br> : 换行标签，是一个单标签 原本单词为：break

文本格式化标签：

< div> 和 < span>标签

< div> 和 < span> 是没有语义的，它们就是一个盒子，用来装内容（布局）

div最显著的特点是一个div只占一行,span都可以放在一行上

图像标签和路径

1、图像标签：< img> 标签用来定义HTML页面中的图像。

图像的插入需要把插入的图片放入与文件相同的路径下
