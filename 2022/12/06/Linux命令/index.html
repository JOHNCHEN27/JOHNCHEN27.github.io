Linux基础

Linux命令基础

ls命令

ls – 查看目录下的内容

语法： ls [-a -l -h ] 指定路径

-a表示： all 查看目录下的全部内容，包含隐藏文件（.开头的文件）

-l表示： 以列表（list）的形式查看目录下的内容

-h表示： 查看目录下文件内容的大小 必须与 -l 一起使用

如果没有选项直接使用ls命令，如: ls 默认是查看当前目录下的内容，不包含隐藏文件，以平铺的方式查看。

nohup java -jar Reggie-1.0-SNAPSHOT.jar &> Reggie.log &

nohup：不挂起后端运行程序， 指定输出日志到Reggie.log文件，最后一个& 表示放在后端运行，退出终端服务器不受影响

停止服务： ps -ef | grep java -jar 查看jar包运行的进程

kill -9 进程id ：杀死进程 停止服务

后台运行reids：src/redis-server redis.conf 前提需要修改配置文件的dae为yes

配置java环境变量

首先将下载好的Linux版本的jdk上传

**解压安装包：tar -zxvf 安装包名 -C /usr/local ** 解压安装包到指定目录下，需要注意的是要将yum命令更新（Centos系统、RedHat系统）

配置环境变量： vim /etc/profile 修改配置文件，在文件末尾写入如下配置：

JAVA_HOME=/usr/local/jdk –设置java环境目录为安装目录

PATH=$JAVA_HOME/bin:$PATH –设置环境变量，追加java目录下的bin目录到环境变量中

重新加载profile文件，使更改的文件立即生效：source /etc/profile

检测是否安装成功：java -version

cd和pwd命令

cd – 切换工作目录

语法： cd 指定路径

相对路径：以当前目录为起点的路径

绝对路径：以根目录为起点的路径 根目录: / (斜杠)

特殊路径: . 表示当前目录 .. 表示上一级工作目录 **~ **表示home目录

pwd：查看当前工作目录

语法： pwd 没有参数

mkdir命令

mkdir – 创建目录

语法： mkdir -p 目录名

当连续创建多个层级目录时，可以使用 -p选项， -p表示创建层级目录

一般情况下 是在当前目录下创建一个目录名

rmdir：删除空目录

语法：rmdir -p 目录名 -p:当子目录被删除后父目录为空则被一并删除

rmdir itcast* ：删除名称以itcast开始的空目录

touch-cat-more命令

touch – 创建文件

语法： touch 文件名

可以用上相对、绝对、特殊路径

cat – 查看文件内容

语法： cat 文件名（Linux路径）

more – 查看文件内容 可以翻页查看

语法： more linux路径

进行翻页查看 用q退出查看

tail 文件名： 查看文件倒数十行内容， tail -20 文件名表示查看文件倒数二十行内容，可以指定参数，tail -f 文件名：表示动态查看文件内容

cp-mv-rm命令

cp （copy）– 复制文件或文件夹

语法： cp [-r] 参数1 参数2 -r表示复制文件夹及其子目录和文件需要复制文件夹时带上-r

参数1表示： 复制的文件名 参数2表示： 复制到哪个路径下 （不写默认当前路径下）

mv（move） – 移动文件或文件夹

语法： mv 参数1 参数2

参数1：被移动的文件或文件夹

参数2：要移动去的地方 （Linux路径）

如果复制文件去的路径不存在则会将 当前的文件名改成路径的那个名称

mv改名：mv hi.txt lncanswer/hello.txt 将hi文件移动到lncanswer文件夹目录并改名为hello

rm（remove） – 删除文件或文件夹或目录

语法： rm [-r -f] 参数1 参数2 参数n

-r： 删除文件夹 -f ： 强制删除文件或文件夹

参数:表示被删除的文件或文件夹， 支持多个用空格隔开， 参数支持通配符 * ，*表示匹配任意内容

注意：rm命令谨慎使用

grep - wc -管道符 |

grep – 从文件中通过关键字过滤文件行

语法： grep [-n] 关键字 linux文件路径

文件路径可以搭配管道符 | 来使用

关键字 建议用 双引号 “ “ 包含起来

-n参数 是表示匹配的关键字有多少行

wc – 统计文件的各类信息

语法： wc [-c -m -l -w] 文件路径

不带选项默认统计： 行数 、单词数、字节数

-c： 字节数 -m：字符数 -l：行数 -w：单词数

文件路径也可以搭配管道符来使用 |

管道符：| 将管道符左边的命令结果 ，作为右边命令的输入

如： 查看text.txt文件下 关键字itheima的行数

cat text.txt | grep “itheima” | wc -l

which-find命令

which – 查看linux命令程序的所在位置

语法： which Linux命令

Linux的命令本质上是一个程序，被存放在某个目录下

find – 查找指定文件

语法： find 起始路径 -name “被查找的文件名”

支持通配符

find / -name “LNC” : 查找根目录下 文件名为 LNC的文件

按文件大小查找： find 起始路径 -size +/- n[KMG]

如： find / -size +100m ：查找根目录下大小大于100M的文件

echo-tail-重定向符

echo – 在命令行输出指定内容

语法： echo 输出的内容

复杂的内容可以用 双引号 “ “ 包围输出

**反引号符： `` 被 反引号符包围的内容，会被作为命令执行，并非普通字符 **

如: echo pwd ； 表示在命令行输出pwd命令的结果

重定向符:

一个右移符: > 将符号左侧命令的结果，覆盖写入到符号右侧指定文件中

两个右移符： >> 将符号左侧命令的结果，追加写入到符号右侧指定文件中

tail – 查看文件尾部内容，并可以持续跟踪

语法： tail [ -f -num ] Linux路径

-f ：持续跟踪 -num ：需要查看的行数 Linux路径：查看的文件名

不写默认查看文件最后十行

需要进入Nginx的sbin目录下面执行

启动Nginx服务命令: ./nginx

Linux防火墙

查看防火墙状态：systemctl status firewalld、firewall-cmd –state

暂时关闭防火墙：systemctl stop firewalld

永久关闭防火墙：systemctl disable firewalld

开启防火墙：systemctl start firewalld

开放指定端口：firewall-cmd –zone=public –add-port=8080/tcp –permanent

关闭指定端口：firewall-cmd –zone=public –remove-port=8080/tcp –permanent

立即生效：firewall-cmd –reload

查看开放的端口：firewall-cmd –zone=public –list-ports

修改权限控制

chmod – 修改文件或文件夹的权限信息

语法： chmod [ -R ] 权限 文件或文件夹 -R表示对文件夹内的全部内容应用同样的操作

chmod 751 text.txt 表示 将text.txt文件的权限修改为 user = 可读可写可执行 group = 可读可执行 other = 可执行

chown – 修改文件、文件夹的所属用户和用户组

语法： chown [-R] [ 用户 ] [ : ] [ 用户组 ] 文件或文件夹

chown命令只适用于root用户执行 ， 普通用户无法修改所属为其他用户或组

Linux实用技巧

强制停止： ctrl + c 某些程序的运行，如果想要强制停止它 或 命令输入错误 可以实用 ctrl + c 强制停止当前

退出或 登出 ： ctrl + d 退出账号的登录 ，退出某些特定程序的专属页面

历史搜索命令： history 查看历史输入过的命令

！ 命令前缀 ： 自动执行上一次匹配前缀的命令 如： ！py ： 从历史命令中由下至上 查找与py相匹配的第一个命令

ctrl + l ：输入内容去匹配命令 如果是需要的内容 回车执行 ； 键盘左右键可以得到此命令

清屏： ctrl + l 或 命令clear 可以清空终端内容
