for循环

语法： for i in range(5): //表示i下标从0到4循环，五次循环

range可以指定输出范围： range(1,5)表示从下标为1的数到下标为4的数，也就是从第二个数到第五个数

range(1,5,2) 表示从第二个数开始到第五个数，隔两个数输出一个.

for循环嵌套:经典案例 输出乘法口诀表

print（）括号中什么也不写 表示换行，默认换行，

while循环

语法：while 变量条件: 指向语句

while循环嵌套：输出九九乘法表

for循环和while循环的区别是： for循环中条件不能取等号，会自动加1，而while循环条件可以取等号，但不会执行加减操作，需要手动添加

for循环可以和while循环嵌套：利用这个也可以输出九九乘法表

与数组不同，列表中的元素可以是不同类型，也从下标开始访问

insert( ) 向列表中插入元素： 插入位置需要指定下标 如： insert(1,”You”) 表示下标为1的位置插入 You

extend( ) 也是在列表中加入元素，但是必须要加入一个序列
